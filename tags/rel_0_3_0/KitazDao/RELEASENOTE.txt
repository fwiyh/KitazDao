0.3.0
* 検証済みDBの追加
Windows上で動くMSAccessを同じ環境にあるIISからODBC経由でアクセスできました。
ただし渡す文字列、SQLファイルをすべてSJIS(SJIS-WIN)にする必要があります。

* 機能修正
・異常発生時にExceptionを投げるようにしました。

* 不具合対応
・設定ファイルのPDOドライバオプションが反映されるようになりました。
連想配列を記述するように、１連想配列をカンマ区切りにして「=>」で配列を作成できます。
例）option="PDO::ATTR_ORACLE_NULLS => PDO::NULL_NATURAL, PDO::ATTR_ERRMODE => PDO::ERRMODE_EXCEPTION"
・READMEの修正をしました。


0.2.0　
* 機能追加
・インスタンス作成時に別の設定ファイルを指定するオプション
KitazDaoディレクトリ直下（libの１つ上の階層）からファイルを取得しにいきます。
例）$dao = new KitazDao("MSectionDao", "KitazDao2.config");

* 不具合対応
SQLファイルを実行する際にパラメータがentityでも配列でもない普通の変数の場合にデータ型を取得できない問題

* 既知の問題
・例外を用意していないため結果がfalseか常にチェックする必要がある。
・設定ファイルのPDO接続のオプション配列が反映されない。
