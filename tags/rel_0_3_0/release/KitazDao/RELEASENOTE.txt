0.2.0　
* 機能追加
・インスタンス作成時に別の設定ファイルを指定するオプション
KitazDaoディレクトリ直下（libの１つ上の階層）からファイルを取得しにいきます。
例）$dao = new KitazDao("MSectionDao", "KitazDao2.config");

* 不具合対応
SQLファイルを実行する際にパラメータがentityでも配列でもない普通の変数の場合にデータ型を取得できない問題

* 既知の問題
・例外を用意していないため結果がfalseか常にチェックする必要がある。
・設定ファイルのPDO接続のオプション配列が反映されない。
